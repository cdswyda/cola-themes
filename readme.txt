## 文章的分类

> 文章的分类，特指网站根目录下的直接文件夹


### 名称不以 _ 开头的

在分类的目录内，创建一个 index.md，然后插入一张图片，以及必要的文字描述，即可形成网站导航上`分类`页面。

### 名称以 _ 开头的

同样，也是创建一个index.md，并且使用[metadata](https://api.bitcron.com/read/metadata)的方式，指定一个标题，这样，这个目录会出现在网站导航中。

在这个目录内，应该再创建子目录，比如以年份划分，再每个子目录内再存储文章，则会自动生成一个特别的页面，比较适合记录自己读过的书、看过的电影。但所有文章总数不超过300每页面。

在每篇文章的 metadata 中，如果指定了一个`link`属性（比如是豆瓣上的一个链接），则会跳转到指定的URL，默认是打开一篇日志的详细页面。
**每篇文章都都应该有插图**，不然会破坏页面的完整性。



## 网站首页

### 封面

在网站根目录下，存放一个`bg.jpg`并同步，即可替换。

### 左上标题

即网站名称，其字体可以在`设置`中选择`CN Font1`。

### 封面上的文字
即网站的二级标题。默认是不换行的，如果需要换行，则使用 HTML 代码 `<br>`即可。其字体可以在`设置`中选择`CN Font2`决定。
